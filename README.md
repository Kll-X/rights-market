# rights-market-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

###代码风格：
1. 为了保护大家的眼睛，尽量缩进使用4个空格
1. 样式名使用a-b-c风格
1. 单位尽量用rem，与@2X的尺寸转化比是100，即设计稿的100px相当于1rem
1. 注意大小写：组件名大写
